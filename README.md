This is a simple web application that allows users to search for meals based on a given ingredient.  
The application uses **TheMealDB public API** to fetch and display recipes.

The project is implemented using **HTML, CSS, and vanilla JavaScript**, without any backend or framework.

---

##  Features
- Search meals by ingredient
- Display a list of meals with images
- View detailed information about a selected meal:
  - Name
  - Country of origin
  - Category
  - Cooking instructions
- Remove meals from the results list
- Add meals to a shopping cart
- View selected meals in a cart overlay
- Loading indicator during API requests

---

##  Technologies Used
- HTML5
- CSS3
- JavaScript (ES6)
- Font Awesome (icons)
- TheMealDB API

---

##  External API
The application uses the public API provided by **TheMealDB**:
- Search meals by ingredient  
  `https://www.themealdb.com/api/json/v1/1/filter.php?i={ingredient}`
- Get meal details by ID  
  `https://www.themealdb.com/api/json/v1/1/lookup.php?i={id}`

---

##  How to Run the Project
1. Download or clone the repository
2. Make sure the following files are in the same folder:
   - `index.html`
   - `style.css`
   - `script.js`
3. Open `index.html` in any modern web browser (Chrome, Edge, Firefox)

No additional setup or server is required.
